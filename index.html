Projects

  • Spam Detection Program

    This code demonstrates the practical application of NLP and machine learning techniques for detecting spam messages. It loads and preprocesses message data, extracts features using CountVectorizer, trains a Naive Bayes classifier, and evaluates its performance using metrics such as accuracy, precision, recall, and F1-score. Additionally, the code visualizes the most frequent words and sentences in spam messages using bar charts. Finally, the code prompts user input, preprocesses it, and leverages the trained classifier to predict whether the input is spam or not.
