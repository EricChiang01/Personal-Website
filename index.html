  • Spam Detection Program

    This program performs text classification to detect spam messages in a dataset of SMS messages. It uses the Naive Bayes algorithm to train a classifier on a subset of the dataset, and then evaluates the performance of the classifier on a test set. It also extracts the most common words and sentences in spam messages from the dataset and plots them using bar charts. Finally, it allows the user to enter a message, preprocesses the input, and then predicts whether the message is spam or not using the trained classifier.
